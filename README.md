# КиберЛаб - Информационный сайт о кибербезопасности

## 📋 О проекте

**КиберЛаб** — это современный информационный сайт, посвященный теме кибербезопасности и защите от вредоносных программ. Проект создан с целью информирования пользователей о различных типах цифровых угроз и методах защиты в 2026 году.

![Версия](https://img.shields.io/badge/версия-2.0.0-blue)
![Лицензия](https://img.shields.io/badge/лицензия-MIT-green)
![Последнее обновление](https://img.shields.io/badge/обновление-март%202026-orange)

### 🎯 Цели проекта:
- Образование пользователей в области кибербезопасности
- Обзор актуальных угроз 2026 года
- Практические рекомендации по защите
- Честное информирование об антивирусном ПО

---

## ✨ Особенности

### 📱 Полностью адаптивный дизайн
- Оптимальное отображение на всех устройствах (смартфоны, планшеты, десктопы)
- Mobile-first подход

### 🎨 Современный UI/UX
- Темно-синяя цветовая схема с акцентными цветами
- Красный — для обозначения опасности
- Зеленый — для элементов защиты
- Четкие гротескные шрифты (Montserrat, Open Sans)

### ✨ Интерактивные элементы
- Анимированный фон с частицами на Canvas
- Плавные появления при скролле
- Ховер-эффекты на карточках
- Плавный скролл до якорей

### 📚 Структурированный контент
- Hero-секция с призывом к действию
- Карточки 8 основных типов угроз
- Статистический блок с актуальными данными
- Чек-лист признаков заражения
- Пошаговая инструкция по спасению
- FAQ с популярными вопросами
- Полноценные юридические страницы

### 🔍 Детальное описание файлов

#### `index.html` - Главная страница
- Hero-секция с заголовком и кнопками
- Блок "Что такое вредоносное ПО"
- Сетка из 8 карточек с типами угроз
- Статистический блок
- Чек-лист симптомов заражения
- Инструкция по защите
- FAQ

#### `privacy.html` - Политика конфиденциальности
- Полное описание сбора и обработки данных
- Таблица с типами собираемых данных
- Информация о cookies
- Права пользователей
- Контактная информация

#### `disclaimer.html` - Юридическое предупреждение
- 11 основных разделов
- Цветные блоки предупреждений
- Таблицы ответственности
- Юридически значимые формулировки

#### `styles.css` - Общие стили
- CSS-переменные для цветовой схемы
- Медиа-запросы для адаптивности
- Стили для всех компонентов
- Анимации и переходы

#### `script.js` - Интерактивность
- Canvas анимация с частицами
- Intersection Observer для анимации появления
- Плавный скролл
- Обработка resize событий
