# КиберЛаб - Информационный сайт о кибербезопасности

## 📋 О проекте

**КиберЛаб** — это современный информационный сайт, посвященный теме кибербезопасности и защите от вредоносных программ. Проект создан с целью информирования пользователей о различных типах цифровых угроз и методах защиты в 2026 году.

![Версия](https://img.shields.io/badge/версия-2.0.0-blue)
![Лицензия](https://img.shields.io/badge/лицензия-MIT-green)
![Последнее обновление](https://img.shields.io/badge/обновление-март%202026-orange)

### 🎯 Цели проекта:
- Образование пользователей в области кибербезопасности
- Обзор актуальных угроз 2026 года
- Практические рекомендации по защите
- Честное информирование об антивирусном ПО

Устройство	Особенности
Десктоп (≥1280px)	Полная верстка, сетка 3-4 колонки
Планшет (768-1279px)	Сетка 2 колонки, уменьшенные отступы
Мобильный (<768px)	Сетка 1 колонка, увеличенный текст, вертикальное расположение
🔧 Кастомизация

css

    --primary-dark: #ваш-цвет;
    --accent-danger: #ваш-цвет;
    --accent-safe: #ваш-цвет;
}
